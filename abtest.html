Listening Test I (AB-Test)

This is the listening test I.

Please choose a clip that you think containing less annoying artifact in each paired clips.

>>> There are 11 paired clips. The number 0 pair is a practice, you can adjust the volume based on these clips and please also answer it on the Google Forms as an exercise.

>>> Make sure you use headphone or earphone while listening(Please do not use loudspeakers). If you need headphones or have any other questions, don't hesitate to ask the developer.

>>> This listening test will take around 5 minutes.
