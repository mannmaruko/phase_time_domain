Listening Test II-3 (MOS)

This is the listening test II-3.

There are 11 groups: One is used for practice, ten are used for test. Each group consists of 4 clips. Each group will be provided a reference, each participant is expected to rate the four clips according to the similarity with the reference. Please rate each clip from 1 to 5.

>>> 5 represents extremely good quality(i.e., almost the same as reference).

>>> 4 represents good quality.

>>> 3 represents fair quality.

>>> 2 represents bad quality.

>>> 1 represents extremely bad quality.

>>> The number 0 group is a practice, you can adjust the volume based on these clips and please also rate them in the Google Forms as an exercise.

>>> Make sure you use headphone or earphone while listening(Please do not use loudspeakers). If you need headphones or have any other questions, don't hesitate to ask the developer.

>>> This listening test will take around 15 minutes.
